# dotfiles

modify paths from .dotfiles/exports for your own setup.

Installation Steps:

```
cd ~
git clone https://github.com/MervanA/dotfiles.git
bash  ~/dotfiles/.dotfiles/install.sh
source ~/.bashrc
```
